# FNIRSI-1013D-WAV-Viewer-in-Excel-VBA
Load, View, Analyze FNIRSI 1013D .wav files in Excel with VBA

(EN) Macro settings: File - Options - Trust Center - Settings... - Macro settings<br>
(x) Disable all Macros with notification<br>
(x) Trust access to the VBA projekt object model<br>
(DE) Macro settings: Datei - Optionen - Trust Center - Einstellungen... - Makroeinstellungen<br>
(x) Deaktivieren von Macros mit Benachrichtigung<br>
(x) Zugriff auf das VBA Projektobjektmodell vertrauen<br>

(EN) VBA Editor: (x) Tools - Options - General - "Break on unhandled errors"<br>
(DE) VBA Editor: (x) Extras - Optionen - Allgemein - "Unterbrechen bei nicht verarbeiteten fehlern"<br>

V1.30: Opens also Data format Peco V0005 and Atlan V0.26d and export CSV<br>
V1.40: FFT implemented for CH1, PlotSize always 1500
